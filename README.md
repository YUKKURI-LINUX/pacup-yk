# pacup
pacup is an automation script to update packages, for Debian/Ubuntu.

## Package Manager Supports
- apt
- Snap
- Flatpak

## Installation
Install:

```make install```

Uninstall:

```make remove```

## Usage
```pacup [-a, -y, -h]```

*Options:*
- a: Run apt only.
- y: Skip confirmations from apt.
- h: Show help.
